## 🏆 Proyectos Destacados

### 🏃‍♂️🤖 **[EnduranceCoach AI]** *(Proyecto Destacado - TFM)*
Plataforma web integral diseñada para revolucionar la gestión deportiva en disciplinas de resistencia. Utiliza modelos generativos de OpenAI para crear planes de entrenamiento y nutrición hiper-personalizados basados en las métricas fisiológicas reales del atleta.

* **Stack:** Java 21, Spring Boot 3.2, Spring AI, Angular 17, PostgreSQL, JWT Security.
* **Hitos técnicos:** * Integración nativa con la API de OpenAI mediante _Prompt Engineering_ avanzado.
  * Arquitectura de seguridad robusta con JSON Web Tokens (JWT).
  * Diseño frontend premium utilizando técnicas de _Glassmorphism_ en CSS puro.
  * *Proyecto Final del Máster de Desarrollo con IA.*

### 📝 **[ExamGenius - Generador de Exámenes]**
Plataforma web para la generación dinámica, gestión y almacenamiento de exámenes educativos por asignaturas y temas.

* **Stack:** Angular 18 (Standalone Components), Node.js, Express, jsPDF.
* **Hitos técnicos:**
  * Exportación directa de exámenes generados a formato PDF (`jsPDF`).
  * Autenticación segura mediante JWT con persistencia de usuarios en backend (File System JSON).
  * Lógica robusta de gestión de estado en cliente (`localStorage`) con recolección de basura para datos corruptos o vacíos.
  * Arquitectura Frontend segura para compilación híbrida / SSR (`isPlatformBrowser`).
